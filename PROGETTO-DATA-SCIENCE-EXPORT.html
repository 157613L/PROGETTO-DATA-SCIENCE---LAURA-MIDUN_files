Esportazioni di prodotti italiani nel mondo

Questo progetto ha lo scopo di rappresentare graficamente landamento che ha subito il fatturato italiano per lesportazione di prodotti nel mondo tra il 1994 e il 2019.

#LIBRERIE UTILIZZATE

library(tidyr)
library (dplyr)
library(ggplot2)
library(graphics)     
library(modelr)
library(purrr)

Inizialmente necessario importare il dataset attraverso il comando read.cvs.

In seguito possibile assegnare al dataset una nuova variabile.

export = read.csv("Export ($ thousand) 1994-2019.csv", header = TRUE, sep = ";")

Il dataset, inoltre, deve essere modificato in modo che si possano avere soltanto i dati che pi interessano per la creazione dei grafici. Quindi, attravarso select (dplyr) vengono selezionate le colonne e con pivot_longer (tidyr) le colonne preesistenti vengono convertite in righe.

tab1 <- select(export, Reporter.Name, Partner.Name, Product.Group, (X1994 : X2019))

tab1 <- pivot_longer( tab1,
                      c('X1994':'X2019'),
                      names_prefix = "X",
                      names_to = "Year",
                      values_to = "Export"
                     )

Ora possibile rappresentare graficamente landamento generale delle esportazioni italiane, dove ogni riga rappresenta un Paese importatore. In questo modo, dal grafico riportato di seguito, possibile notare un andamento non regolare nel tempo e non simile per tutti gli Stati presi in considerazione.
Questa rappresentazione grafica non aiuta, per, a capire chiaramente nel dettaglio come la situazione si sviluppata nel corso degli anni.

ggplot(tab1, aes(x = as.numeric(Year), y = as.numeric(sub(",", ".", Export, fixed = TRUE)))) +
  geom_line(aes(group = Partner.Name), alpha = 1/2) +
  theme_classic() +
  xlab("Anni") +
  ylab("Esportazioni") +
  scale_x_continuous(breaks = c(1994, 2000, 2005, 2010, 2015, 2019),
                     limits = c(1994, 2019)) +
  ggtitle("Esportazioni (US$) 1994-2019")

Quindi, viene preso in esame uno dei Stati pi rilevanti, gli Stati Uniti, ovvero dove lItalia ha conseguito un profitto maggiore e viene tracciarto il suo grafico. possibile notare che in generale landamento crescente ma nellarco di tempo tra il 2005 e 2010 si verifica unimportante oscillazione.

us_exp <- filter(tab1, Partner.Name == "United States")

us_exp %>%
ggplot(aes(x = as.numeric(Year), y = as.numeric(sub(",", ".", Export, fixed = TRUE)))) + 
  geom_line() + 
  xlab("Anni") +
  ylab("Esportazioni") +
  scale_x_continuous(breaks = c(1994, 2000, 2005, 2010, 2015, 2019),
                     limits = c(1994, 2019)) +
  ggtitle("Italia - USA") 

us_exp_lt <- lm((as.numeric(sub(",", ".", Export, fixed = TRUE))) ~ (as.numeric(Year)), data = us_exp)

us_exp %>% 
  add_predictions(us_exp_lt) %>%      
  ggplot(aes((as.numeric(Year)), pred)) + 
  geom_line() +
  xlab("Anni") +
  scale_x_continuous(breaks = c(1994, 2000, 2005, 2010, 2015, 2019),
                     limits = c(1994, 2019)) +
  ggtitle("Andamento lineare")

Ora vengono calcolati i resuidi sul Paese appena considerato per poi adattare la formula a tutti gli altri Stati.

#RESIDUI 

us_exp %>% 
  add_residuals(us_exp_lt) %>% 
  ggplot(aes((as.numeric(Year)), resid)) + 
  geom_hline(yintercept = 0, colour = "white", size = 3) + 
  geom_line() +
  xlab("Anni") +
  ylab("Residui") +
  scale_x_continuous(breaks = c(1994, 2000, 2005, 2010, 2015, 2019),
                     limits = c(1994, 2019)) +
  ggtitle("Residui USA")

Attraverso la funzione nest() (tidyr) vengono annidati i dati creando un dataset per ciascun Paese.

by_partner_name <- tab1 %>% 
  group_by(Partner.Name) %>% 
  nest()                          
partner_name_model <- function(df) {
  lm((as.numeric(sub(",", ".", Export, fixed = TRUE))) ~ (as.numeric(Year)), data = df)
}

Con mutate (dplyr) viene creata una nuova variabile mentre con map (purrr) si applica by_partner_name ad ogni elemento.

by_partner_name <- by_partner_name %>% 
  mutate(model = map(data, partner_name_model))

Ora possibile creare un grafico nel quale vegono tracciati i residui.

resids %>% 
  ggplot(aes((as.numeric(Year)), resid)) +
  geom_line(aes(group = Partner.Name), alpha = 1 / 3) + 
  geom_smooth(se = FALSE, color = "red") +       
  theme_classic() +
  xlab("Anni") +
  scale_x_continuous(breaks = c(1994, 2000, 2005, 2010, 2015, 2019),
                     limits = c(1994, 2019)) +
  ggtitle("Residui")

Dal grafico si pu notare che lItalia in alcuni Stati ha esportato meno prodotti rispetto che in altri. Per capire come si evoluta nel tempo la situazione per questi ultimi applichiamo filter (dplyr).

tab2 <- filter(tab1, (as.numeric(sub(",", ".", Export, fixed = TRUE))) <= 1000)
